# Internal

Operations areas — the business of running your business.

Examples:
- `Sales/` — pipeline, proposals, prospects
- `Hiring/` — JDs, interviews, candidates
- `Finance/` — budgets, invoicing, expenses
- `Brand/` — assets, voice, content guidelines
- `Training/` — SOPs, onboarding docs

Each area is a folder. Inside, free-form structure that suits the area.
