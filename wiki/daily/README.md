# Daily

One file per day, named `YYYY-MM-DD.md`.

Format suggestion (use `_templates/daily-template.md`):

```markdown
# 2026-04-30

## Focus today
- ...

## Done
- ...

## Notes / thoughts
- ...

## Tomorrow
- ...
```

Daily notes sit cross-project. They're the chronological view across all your work, useful for "what was I doing on Tuesday?" or weekly reviews.

Don't force it. Skip days when nothing's worth recording.
