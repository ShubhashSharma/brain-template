# Daily

One file per day, named `YYYY-MM-DD.md`. Use `_templates/operator-daily.md`.

---

## The morning ritual (10 minutes)
1. Open today's daily note (Templater creates it on hotkey).
2. Pull yesterday's numbers from each channel admin (or run `/refresh-yesterday` if you've wired up MCP).
3. Update OOS / low-stock watch list.
4. Write 3 wins, 3 fires.
5. List today's decisions to make.

`EXAMPLE-2026-05-09.md` - a populated daily note showing what the morning ritual looks like in actual use.

---

## Why this folder beats a Notion daily
- Plain text. Searchable forever. No vendor lock-in.
- Linked. Every SKU you mention is a backlink.
- Claude Code can read 90 days of dailies in one prompt.

---

## What you'll notice after 30 days
The same fires keep recurring. That's a signal to write a playbook or hire someone.
