# Clients

One folder per client. Use this structure for each:

```
ClientName/
├── MOC.md           ← Map of Content. The home page for this client.
├── context.md       ← Who they are, what they do, current priorities, key people.
├── sessions/        ← One file per work session. YYYY-MM-DD-topic.md
├── tasks/           ← Open work items. todo.md is the running list.
└── lessons/         ← Patterns / mistakes / things worth remembering for next time.
```

Use `_templates/client-MOC-template.md` to create a new client folder.

Link liberally to people in `wiki/entities/people/` and companies in `wiki/entities/companies/`.
