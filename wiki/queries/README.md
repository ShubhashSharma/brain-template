# Queries

Saved [Dataview](https://blacksmithgu.github.io/obsidian-dataview/) queries. Lets you build dashboards inside Obsidian.

Examples:
- `open-tasks-all-clients.md` — every uncompleted task across `wiki/clients/*/tasks/`
- `recent-sessions.md` — last 7 days of session notes, any folder
- `lessons-this-quarter.md` — every lesson logged in the last 90 days

Dataview reads metadata from frontmatter (`tags`, `client`, `status`) and tasks (`- [ ]` syntax) across the whole vault.

Start simple. One query that answers a real question is worth ten elaborate ones.
