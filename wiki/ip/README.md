# IP

Trademarks, patents, compliance docs, lab tests. One page per registration or document.

---

## Why this folder matters more than you think
The first time a competitor scrapes your listing, the first time a buyer asks "show me your CPSIA cert", the first time Amazon suspends you for a claim, you'll wish this folder existed.

---

## Suggested naming
- `trademark-{brand}-{jurisdiction}.md` - e.g. `trademark-acme-uspto.md`
- `patent-{title}.md`
- `compliance-{cert-type}-{sku}.md` - e.g. `compliance-cpsia-acme-baby-shampoo.md`
- `lab-test-{sku}-{type}.md`

---

## Always capture
- Registration / certificate number
- Jurisdiction
- Filing date, expiry / renewal date
- Where the original PDF lives (Drive / Dropbox link is fine)
- Linked SKU(s)

A 5-minute page now is a 5-figure save later.
