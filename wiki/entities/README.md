# Entities

People and companies you interact with. Two subfolders:

```
entities/
├── people/           ← One file per person. Notes, links, interactions, context.
└── companies/        ← One file per company. What they do, who works there, status.
```

Why separate? So you can `[[link]]` to a person from anywhere — a client session, a meeting note, a daily — and see all their mentions in the backlinks panel. Same for companies.

The graph view becomes your network map.
